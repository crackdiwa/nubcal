# 🍚 นับแคล — NubCal

แอพนับแคลอรี่ภาษาไทย ใช้งานง่าย เปิดจากมือถือได้เลย ไม่ต้องติดตั้ง
**ถ่ายรูปอาหารแล้วให้ AI คำนวณ** แคล · โปรตีน · คาร์บ · ไขมัน · โซเดียม · น้ำตาล ให้อัตโนมัติ

> **เปิดใช้เลย:** https://crackdiwa.github.io/nubcal/

## ทำอะไรได้บ้าง

- 📷 **ถ่ายรูปอาหาร** — AI (Gemini) ดูรูปแล้วประเมินสารอาหารทุกอย่างในจาน
- 🏷️ **สแกนบาร์โค้ด** — ขนม เครื่องดื่ม ของในเซเว่น ดึงข้อมูลจาก Open Food Facts
- ✏️ **พิมพ์เอง** — มีเมนูอาหารไทยยอดฮิตกว่า 110 รายการในตัว หรือพิมพ์อะไรก็ได้ให้ AI ประเมิน
- 🎯 **เป้าหมายส่วนตัว** — กรอกน้ำหนัก ส่วนสูง อายุ กิจกรรม แล้วเลือกเป้าหมาย (ลดไขมัน / รักษาน้ำหนัก / เพิ่มกล้ามเนื้อ / เพิ่มน้ำหนัก) แอพคำนวณแคลที่ควรกินต่อวันให้ พร้อมหัก/บวก calorie deficit อัตโนมัติ
- 📊 **สรุปรายวัน + กราฟย้อนหลัง** 7/30 วัน
- 👤 **ระบบผู้ใช้ในเครื่อง** — สมัคร/ล็อกอิน แยกข้อมูลแต่ละคน ข้อมูลอยู่ในเครื่องคุณเท่านั้น

## เริ่มใช้งาน (3 ขั้น)

1. เปิด https://crackdiwa.github.io/nubcal/ → กด **สมัครใหม่**
2. กรอกข้อมูลร่างกายและเป้าหมาย — แอพจะบอกเลยว่าควรกินวันละกี่แคล
3. (ถ้าอยากใช้ AI ดูรูป) ขอ **Gemini API key ฟรี** ที่ [aistudio.google.com/apikey](https://aistudio.google.com/apikey) แล้ววางในหน้า **โปรไฟล์** — ครั้งเดียวจบ

> 💡 เพิ่มลงหน้าจอโฮมมือถือ (Add to Home Screen) จะใช้เหมือนแอพจริง

## วิธีคำนวณ

- **BMR** ตามสูตร Mifflin–St Jeor แล้วคูณระดับกิจกรรมเป็น **TDEE**
- ลดไขมัน −500 แคล/วัน (≈ ลด 0.5 กก./สัปดาห์) · เพิ่มกล้าม +300 · เพิ่มน้ำหนัก +500
- โปรตีนตั้งตามเป้าหมาย (1.4–1.8 ก./กก.) ไขมัน ~27.5% ของพลังงาน ที่เหลือเป็นคาร์บ
- เพดานโซเดียม 2,000 มก. และน้ำตาลไม่เกิน 10% ของพลังงาน (ตาม WHO)

## เทคนิค

ไฟล์เดียวจบ (`index.html`) — HTML + CSS + JavaScript ล้วน ไม่มี build ไม่มี backend
ข้อมูลทั้งหมดเก็บใน `localStorage` ของเบราว์เซอร์ · AI ใช้ Gemini API (key ของผู้ใช้เอง) · บาร์โค้ดใช้ BarcodeDetector + Open Food Facts

## ข้อควรรู้

- ตัวเลขสารอาหารเป็น **ค่าประมาณ** ไม่ใช่คำแนะนำทางการแพทย์
- ข้อมูลเก็บในเครื่อง/เบราว์เซอร์ที่ใช้ ถ้าล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ข้อมูลไม่ตามไป
